# Python Assignment 3 (Variant B)

Этот проект представляет собой итоговую работу, объединяющую Практические работы №4, №5 и №6. Программа анализирует данные студентов из CSV-файла, используя объектно-ориентированное программирование (ООП), обработку исключений и продвинутые функции Python.

## 📋 Описание проекта

Программа выполняет следующие задачи:
1.  *FileManager*: Проверяет наличие исходного файла и создает структуру папок.
2.  *DataLoader*: Загружает данные из students.csv с использованием csv.DictReader.
3.  *DataAnalyser (Вариант Б)*: Подсчитывает количество студентов по странам и определяет ТОП-3 страны с наибольшим числом учащихся.
4.  *ResultSaver*: Сохраняет результаты анализа в формате JSON.

## 🛠 Технологии и требования

В проекте реализованы следующие критерии:
- *Модуль OS*: Проверка путей и создание директорий.
- *ООП*: Весь код разбит на 4 независимых класса (FileManager, DataLoader, DataAnalyser, ResultSaver).
- *Обработка исключений*: Использование блоков try-except для предотвращения ошибок при чтении/записи.
- *Lambda & Filter*: Использование анонимных функций для фильтрации данных и сортировки результатов.
- *Формат данных*: Чтение из CSV и экспорт в JSON.
